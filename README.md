# Installation: Virtual Enviorment

### Setting Up Your Virtual Enviorment

    cd Desktop/
    python3 -m venv wsb
    cd wsb
    source bin/activate

### Installing With Git

    cd ../
    git clone git@github.com:abraraltaf92/Whatsapp-Spam-Bot.git

### Installing The Packages

    pip install -r requirements.txt
